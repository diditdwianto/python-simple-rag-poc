# Help — What I Can Help You With (Knowledge Base Index)

This is the help page and topic index for this assistant. Use it when you want to
know what I can do or what subjects I know about.

If you ask any of these — "help", "what can you help me with", "what can I ask
you", "what do you know", "what knowledge do you have", "what topics do you have",
"what subjects are available", "list the available topics", "what is in your
knowledge base", or "what can you do" — the answer is the list of topics below.

## Topics available in my knowledge base

I can answer questions grounded in the following knowledge bases. These are the
subjects and topics currently available in the database:

- **RAG (Retrieval-Augmented Generation)** — what RAG is, how the index and
  retrieval phases work, chunking, embeddings, and grounded answer generation.
- **Redis** — the open-source in-memory data store, Redis Stack, vector search,
  and using Redis as a vector database.
- **RainbowCandy** — the chewy color-changing candy by Lumina Sweets: its
  flavors, colors, history, and slogan.
- **Alien Snacks (Galactic Snackpedia)** — a field guide to popular snacks from
  across the galaxy and the Outer Rim.

Ask me a question about any of these topics and I will answer using the stored
documents. If your question falls outside these topics, I will tell you I don't
have enough information to answer.
